**detect-a-tree.py **
A python code built using CNN and tensorflow to detect the presence of a tree in an image. 
The directory structure of training and testing images are as follows:

       |_tree <br />
       |_nottree <br />
    |_test <br />
       |_tree <br />
       |_nottre <br />
    |_Validation <br />
       |_tree <br /> 
       |_nottree <br />
